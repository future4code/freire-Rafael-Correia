Criptógrafo

Um Decodificador de textos/mensagens funcional: o usuário digita um texto e pode escolher entre criptografar ou descriptografar a mensagem.

A lógica é feita em JavaScript e basicamente altera as palavras do texto, modificando certas letras para que o texto fique incompreensível. Algo similar é feito para decodificar.

A estrutura e estilização da página é feita de maneira responsiva e acessível com Html e CSS, usando tags semânticas, evitando alto contraste de cores e usando medidas responsivas para mobile.

Tecnologias ultilizadas:

  • JavaScript
  • HTML5
  • CSS3
